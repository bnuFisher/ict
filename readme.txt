爬虫及部分文本挖掘分析代码

说明：

1. news_spider.py是我自己写的一个很简单的单线程爬虫，可以直接用来爬取中国教育信息化网站上的新闻报道。
   注：没有使用现有的爬虫框架；由于数据量不算大，也没有使用重试机制。
  
2. 爬去中国知网的论文信息前，需要先获得相关论文的url队列，知网搜索后的论文链接在静态页面中是看不到的，
   可以自己写代码解析js的返回值然而再保存url，也可以移步专门的爬虫工具批量获取，我用的后者，可参考集搜客的教程 -- http://www.gooseeker.com/ 
   获取了url队列后，可直接解析批量页面内容，爬去保存需要的信息。代码见cnki_xxx.py。
